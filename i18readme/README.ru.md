# i18readme

Переводи свой `README.md` на любой язык одной командой. Работает с Claude или GPT-4o.

Языковая панель выше внедряется автоматически в каждый файл. Когда твой README меняется, пересоздаются только устаревшие переводы.

---

## Установка

```bash
npm install -g i18readme
```

Требуется Node.js 18+.

---

## Быстрый старт

```bash
# 1. Перейди в папку проекта
cd my-project

# 2. Инициализация (интерактивно — спросит языки, папку, файлы README)
i18r init

# 3. Добавь свой API ключ (Claude или OpenAI)
i18r config

# 4. Переводи
i18r sync
```

Вот и всё. Твои переводы появятся в `i18readme/README.ru.md`, `i18readme/README.de.md` и т. д.

---

## Команды

```
i18r init              Инициализировать проект (интерактивно)
i18r sync              Перевести недостающие и устаревшие файлы
i18r sync --force      Пересоздать все переводы
i18r status            Показать, что актуально, а что нет
i18r config            Добавить или изменить API ключ (интерактивно)
i18r config show       Показать сохранённую конфигурацию
```

---

## Как это работает

**`i18r init`** сканирует проект в поиске README файлов и создаёт `.i18readme.json`:

```json
{
    "langs": ["ru", "de", "zh"],
    "dir": "i18readme",
    "readmes": ["README.md"]
}
```

**`i18r sync`** проверяет время изменения файлов — если твой `README.md` не менялся с момента последней синхронизации, перевод пропускается. Пересоздаются только действительно устаревшие переводы.

**Каждый переведённый файл** получает панель выбора языка в начале с ссылками на все остальные языки.

## Провайдеры ИИ

| Провайдер          | Модель          | Как получить ключ                                      |
| ------------------ | --------------- | ------------------------------------------------------ |
| Claude _(по умолч.)_ | claude-opus-4-6 | [console.anthropic.com](https://console.anthropic.com) |
| OpenAI             | gpt-4o          | [platform.openai.com](https://platform.openai.com)     |

Запусти `i18r config`, чтобы выбрать провайдера и сохранить ключ. Ключ хранится в `~/.i18readmerc` на твоей машине — никогда в проекте.
