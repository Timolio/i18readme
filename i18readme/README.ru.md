# i18readme

Переводите свой `README.md` на любой язык одной командой. Работает на Claude или GPT-4o.

Языковая панель выше автоматически добавляется в каждый файл. При изменении README заново генерируются только устаревшие переводы.

---

## Установка

```bash
npm install -g i18readme
```

Требуется Node.js 18+.

---

## Быстрый старт

```bash
# 1. Перейдите в свой проект
cd my-project

# 2. Настройте (интерактивно — спросит языки, папку, README-файлы)
i18r init

# 3. Добавьте API-ключ (Claude или OpenAI)
i18r config

# 4. Переведите
i18r sync
```

Готово. Переводы появятся в `i18readme/README.ru.md`, `i18readme/README.de.md` и т.д.

---

## Команды

```
i18r init              Настроить проект (интерактивно)
i18r sync              Перевести отсутствующие и устаревшие файлы
i18r sync --force      Перевести всё заново
i18r status            Показать, что актуально, а что устарело
i18r config            Добавить или изменить API-ключ (интерактивно)
i18r config show       Показать сохранённую конфигурацию
```

---

## Как это работает

**`i18r init`** сканирует проект на наличие README-файлов и создаёт `.i18readme.json`:

```json
{
    "langs": ["ru", "de", "zh"],
    "dir": "i18readme",
    "readmes": ["README.md"]
}
```

**`i18r sync`** проверяет время изменения файлов — если `README.md` не менялся с последней синхронизации, перевод пропускается. Заново переводится только то, что действительно устарело.

**Каждый переведённый файл** получает языковую панель в начале со ссылками на все остальные языки.

## AI-провайдеры

| Провайдер               | Модель          | Как получить ключ                                      |
| ----------------------- | --------------- | ------------------------------------------------------ |
| Claude _(по умолчанию)_ | claude-opus-4-6 | [console.anthropic.com](https://console.anthropic.com) |
| OpenAI                  | gpt-4o          | [platform.openai.com](https://platform.openai.com)     |

Запустите `i18r config`, чтобы выбрать провайдера и сохранить ключ. Ключ хранится в `~/.i18readmerc` на вашей машине — никогда в проекте.
